Git and GitHub:
Git is a application which is used for version control. git is a centralised version control system. git keeps track of all your version changes and updates.
git will contain three phases
     Working phase/Working area:  git init
     Staging phase/Staging area:  git add .
     Local Repository:  git commit -m "message"
